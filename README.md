# Employee-Management-System
Employee management system developed using Java and MySQL.

## Overview

The Employee Management System is a Java-based application developed to manage employee records efficiently. It provides functionalities to add, update, delete, and view employee information, helping organizations maintain employee data in an organized manner.

## Features

* Add new employee records
* View employee details
* Update employee information
* Delete employee records
* Search employee records
* User-friendly interface
* Data management and validation

## Technologies Used

* Java
* IntelliJ IDEA
* MySQL (if used)
* JDBC (if used)

## Project Structure

* Employee.java – Employee model class
* Main.java – Application entry point
* Other supporting classes for employee management operations

## How to Run

1. Clone the repository.
2. Open the project in IntelliJ IDEA.
3. Configure the database connection (if applicable).
4. Build and run the project.
5. Use the menu options to manage employee records.
